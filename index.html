Bienvenue au Château de Couturelle

Dans ce lieu chargé d'histoire et d'élégance, nous vous invitons à vivre une aventure unique.
Cette chasse au trésor a été conçue spécialement pour vous, Stéphanie et Loïc, afin de découvrir ensemble le code secret d'une boîte à clé contenant la clé qui vous ouvrira la porte de votre chambre nuptiale.

Comment jouer ?

  1. Résolvez les énigmes révélées par les tags RFID disséminés dans la salle de réception.
  2. Chaque indice vous rapprochera du code secret.
  3. Travaillez ensemble, prenez votre temps, et profitez de chaque instant.
  4. À la fin, entrez le code dans la boîte à clé pour récupérer la clé de votre chambre.
Vous pouvez le faire, vous êtes des supers Héros !
Bon courage et que l’aventure commence !
